# DynaMER-ADF v0.1.0

Initial public software release of DynaMER-ADF.

This release includes reproducibility code, configuration files, split-generation scripts, temporal feature preparation workflows, baseline training scripts, DynaMER-family training scripts, ablation workflows, nested LOSO evaluation, physiological separability diagnostics, and result-generation utilities.

Raw datasets, generated feature arrays, trained checkpoints, and private local path files are not included.

This is a software-only release. Manuscript-linked citation metadata will be added after publication.
